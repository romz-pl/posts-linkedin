# SU2: The Open-Source CFD Code


**SU2** is an open-source collection of C++ and Python software tools used to analyze partial differential equations (PDEs) and PDE-constrained optimization problems on unstructured meshes. These tools employ state-of-the-art numerical methods. Thanks to the efforts of users and developers worldwide, SU2 has become a well-established, widely applicable tool in the aeronautical, automotive, shipbuilding, and renewable energy industries, to name a few.

Although computational analysis tools have revolutionized the way we design engineering systems, most established codes are proprietary, unavailable, or too expensive for many users. The SU2 team is changing this by making multiphysics analysis and design optimization software freely available and inviting everyone to participate in its creation and development.
